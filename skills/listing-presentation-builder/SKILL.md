# Listing Presentation Builder

**Coming with Episode 9 — [Listing Presentation — From Comps to Brief](https://youtube.com/@theprosperityagents)**

The full SKILL.md drops the day this episode airs. Preview:

- Input: property data + 3-6 comps + seller's stated motivation
- Output: 8-section listing-presentation brief
  1. Cover / agent intro (uses [`canva-foundation`](../canva/foundation/) Brand Kit)
  2. Market context (1 paragraph, FH-clean)
  3. Comp summary table + narrative
  4. Price-positioning rationale
  5. Marketing plan (uses [`canva-just-listed`](../canva/just-listed/), [`canva-open-house`](../canva/open-house/))
  6. Pricing recommendation (range + strategic notes)
  7. Timeline + next steps
  8. Why-us (specific differentiators, never knocks competitors)

Output is text-only — pair with [`canva/listing-presentation`](../canva/listing-presentation/) for the slide visuals.

References [`fair-housing-overlay`](../fair-housing-overlay/) and [`cma-narrative`](../cma-narrative/) (the comp section reuses CMA logic).

Subscribe to [@theprosperityagents](https://youtube.com/@theprosperityagents) for the drop.
