# Canva Coming Soon

**Coming with Episode 15 — [Coming Soon Teaser](https://youtube.com/@theprosperityagents)**

The full SKILL.md drops the day this episode airs. Preview:

- Input: pre-listing data + expected MLS go-live date + seller permission to tease
- Output:
  - IG square teaser caption (NEVER says "available now" or "showings available" — those are MLS-rule violations)
  - FB teaser post
  - TikTok teaser caption (10-15 second hook script)
  - Canva briefs: `CS-Square-Mystery`, `CS-Story-Reveal-Date`

Compliance: MLS rules in most states require the listing to be in the MLS before any public marketing claims availability. This skill enforces "coming soon" framing only — never "available", "showings open", or "call now to schedule."
