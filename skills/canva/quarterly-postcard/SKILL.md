# Canva Quarterly Postcard

**Coming with Episode 18 — [Quarterly Postcard for Geographic Farming](https://youtube.com/@theprosperityagents)**

The full SKILL.md drops the day this episode airs. Preview:

- Input: target neighborhood + last quarter's market data (sold count, median, DOM)
- Output: 4 Canva briefs (one per season) for folder `07-Quarterly-Postcards/`
  - Q1: Year-end recap + spring forecast
  - Q2: Spring market snapshot + buyer demand signals
  - Q3: Summer activity + season-transition framing (no school refs)
  - Q4: Year-to-date + holiday-season inventory note
- Front + back layouts for USPS EDDM 6.5×9 postcard standard
- FH-strict: never references schools, demographics, or "family-friendly" framing
- Always positive about the neighborhood

References [`fair-housing-overlay`](../../fair-housing-overlay/), [`canva-foundation`](../foundation/).
