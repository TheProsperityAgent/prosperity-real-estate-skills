# Canva CMA Cover

**Coming with Episode 17 — [CMA Cover That Sells the Meeting](https://youtube.com/@theprosperityagents)**

The full SKILL.md drops the day this episode airs. Preview:

- Input: output from [`cma-narrative`](../../cma-narrative/) + subject property photo + 3-6 comp photos
- Output: 4 Canva slide briefs for folder `06-CMA-Covers/`
  - `CMA-01-Cover` (subject property + agent intro)
  - `CMA-02-Comps-Grid` (visual layout for 3-6 comps with photo + key stats)
  - `CMA-03-Price-Range` (graphic: low/recommended/high with rationale)
  - `CMA-04-Next-Steps` (sign-now or revisit-in-30 paths)

Each brief specifies exact text replacements + photo crop + which Brand Kit color each section uses.

References [`fair-housing-overlay`](../../fair-housing-overlay/), [`canva-foundation`](../foundation/), [`cma-narrative`](../../cma-narrative/).
