# Canva Just Sold

**Coming with Episode 14 — [Just Sold + Testimonial Request](https://youtube.com/@theprosperityagents)**

The full SKILL.md drops the day this episode airs. Preview:

- Input: closed listing data + your role (listing or buyer side) + permission status to publicize
- Output:
  - IG square caption (≤2200 chars, celebrates closing, FH-clean)
  - FB post (≤700 chars, ends with congratulations to client by first name + permission)
  - TikTok caption (≤300 chars)
  - Canva brief for `JS-Square-Photo-Forward` template
  - Canva brief for `JS-Story-Photo-Forward` template
  - Testimonial-request DM (post-close, 3-5 sentences, asks one specific question)
  - Testimonial-request email (5-7 sentences, includes Google review link + Zillow link)

Compliance: never includes financial details (final sale price requires explicit client opt-in per FHA + state rules). Defaults to "sold" without price unless agent confirms permission.
