# Quick Start — 3 Wins for Realtors

## When to use this skill
You just installed Claude. You want three workflows running today, not next week. You don't want to read a tutorial — you want copy-paste-go.

## What it does
Three prompts. One paragraph each. Compliant from the first character: no school references, no scarcity language, no fair-housing violations, always positive.

## Prompt 1 — Listing Description

```
You're writing a [CITY] [STATE] listing description for me, an agent.
Tone: warm, specific. NEVER use "must-see" or "won't last" or any cliché.
145 words max. The home is at [ADDRESS], [BEDS] bed [BATHS] bath, [SQFT]
sqft, on [LOT] acres, built [YEAR]. Standout features: [3 BULLETS]. End
with a line about [CITY] life that doesn't mention schools.
```

**Why these constraints:**
- Banned clichés ("must-see", "won't last") signal low-quality content to search engines.
- The school exclusion is a fair-housing rule — we never sort buyers by school perception.
- 145-word cap matches the optimal MLS field length.

## Prompt 2 — Counter-Offer Reply

```
I'm a real estate agent. My buyer offered $[X] on a list price of $[Y].
The seller countered at $[Z] with [TERMS]. Write a reply email to the
listing agent that holds my position on price but signals flexibility on
close date / inspections / appliances. Tone: collegial, no pressure,
respect their professionalism.
```

**Why this works:**
- The reply email is half the negotiation. Most agents under-write it.
- "Collegial, no pressure" matters in small markets where you'll see the same listing agents every week.
- Hold price, flex on close-date / appliances — that's how a 20-year agent writes.

## Prompt 3 — Three Social Posts From Today's MLS Pull

```
I'm a real estate agent in [CITY] [STATE]. Here are today's [N] new
listings under $[BAND]: [PASTE]. Write 3 social posts:
  - Post 1: Instagram caption ≤2200 chars, hook + 5 hashtags, no links
  - Post 2: Facebook ≤700 chars, conversational, ask a question
  - Post 3: TikTok caption ≤300 chars, punchy, 5 hashtags
Frame: positive, never knock anywhere or anyone. No school rankings.
No "won't last."
```

**Why these constraints:**
- IG 2200 / FB 700 / TikTok 300 = the actual Notion-rule character caps for compliance.
- "Always positive" and "no school rankings" are fair-housing requirements.
- 5 hashtags hits the sweet-spot for reach on every major platform in 2026.

## Compliance notes

- **Fair Housing:** never references schools, never describes neighborhoods by demographic.
- **TCPA:** these prompts produce content for owned channels (your social, your listings) — not unsolicited outreach.
- **MLS rules:** banned clichés ("must-see", "won't last") protect against MLS data-quality penalties.
- **Always positive:** never knocks a town, builder, neighborhood, competing platform, or competing agent.
